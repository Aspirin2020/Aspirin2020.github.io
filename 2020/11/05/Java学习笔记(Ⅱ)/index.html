数据型直接量

  • 声明一个常量（静态变量）

各种数据类型同C，应该是一项世界标准。操作符优先级也与C相同。

可以通过加括号强制一个变量的类型：

  • 整形直接量

怎么直接输出其他进制的直接量，比如displays 0B1111？

  • 浮点型直接量

浮点型直接量带小数点，默认为double型。在数字后加F或f可表示为float直接量；在数字后加D或d可表示为double直接量。

double小数点后16位，float小数点后8位。

  • 科学计数法

139814可记为1.39814E5、1.39814e5、1.39814E+5、1.39814e+5；0.00123可记为1.23E-3、1.23e-3.
